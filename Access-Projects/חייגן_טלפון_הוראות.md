# חייגן טלפון ל-Access - הוראות התקנה ושימוש

## 📋 מבוא
חייגן טלפון מתקדם לסביבת Access עם ממשק נוח וידידותי למשתמש.

## 🚀 התקנה

### 1. יצירת טבלאות
פתח את Access והפעל את הסקריפט `PhoneDialerTables.sql`:
1. פתח חלון SQL חדש (Create → Query Design → SQL View)
2. העתק והדבק את כל הקוד מהקובץ
3. הפעל את הסקריפט (Run)

### 2. ייבוא הקוד VBA
1. פתח את עורך ה-VBA (Alt+F11)
2. ייבא את הקבצים הבאים:
   - `PhoneDialerModule.bas` - מודול הפונקציות
   - `PhoneDialerForm.frm` - טופס החייגן

### 3. הגדרות אבטחה
ודא שההגדרות מאפשרות הפעלת קוד VBA:
- File → Options → Trust Center → Trust Center Settings
- הפעל "Enable all macros" או "Enable macros"

## 🎞️ ממשק המשתמש

### רכיבי הטופס:
- **תצוגת מספר טלפון** - מציגה את המספר שהוזן
- **לוח חיוג** - מקשים 0-9, *, #
- **כפתורי שליטה**:
  - 📞 חיוג - ביצוע חיוג
  - 📞 ניתוק - סיום שיחה
  - 🗑️ ניקוי - מחיקת מספר
  - ⬅️ Backspace - מחיקת ספרה אחרונה
- **חיוג מהיר** - 5 כפתורים למספרים נפוצים
- **רשימת אנשי קשר** - רשימת אנשי הקשר מהטבלה
- **היסטוריית שיחות** - שיחות אחרונות
- **סטטוס שיחה** - הצגת מצב החיוג

## 📱 שימוש בחייגן

### חיוג בסיסי:
1. הזן מספר טלפון באמצעות לוח החיוג
2. לחץ על כפתור "חיוג"
3. המערכת תחייג את המספר

### חיוג מהיר:
- לחץ על אחד מכפתורי החיוג המהיר (1-5)
- המערכת תחייג את המספר המוגדר מראש

### חיוג מאנשי קשר:
1. בחר איש קשר מהרשימה
2. לחץ פעמיים על האיש קשר
3. המערכת תחייג את מספר הטלפון

## 👥 ניהול אנשי קשר

### הוספת איש קשר:
1. לחץ על "הוסף איש קשר"
2. הזן שם ומספר טלפון
3. הוסף אימייל והערות (אופציונלי)
4. לחץ אישור

### חיפוש איש קשר:
1. לחץ על "חיפוש איש קשר"
2. הזן שם לחיפוש
3. המערכת תמצא את האיש קשר ותציג את מספר הטלפון

## 🔧 הגדרות מתקדמות

### אימות מספרי טלפון:
המערכת מאמתת מספרי טלפון ישראליים:
- קווים: 02, 03, 04, 08, 09 + 7 ספרות
- ניידים: 05X + 7 ספרות (050, 052, 053, 054, 055, 058)
- חירום: 100, 101, 102, 103

### עיצוב מספרים:
המערכת מעצבת מספרים אוטומטית:
- 9 ספרות: XX-XXX-XXXX
- 10 ספרות: XXX-XXX-XXXX (ניידים) או XX-XXX-XXXX (קווים)

## 📝 היסטוריית שיחות

המערכת רושמת אוטומטית:
- מספר טלפון
- שם איש קשר (אם מזוהה)
- תאריך ושעה
- משך שיחה (אם זמין)
- סוג שיחה (יוצאת/נכנסת/לא נענתה)
