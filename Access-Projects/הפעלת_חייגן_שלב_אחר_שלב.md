# הפעלת חייגן טלפון ב-Access - שלב אחר שלב

## 🚀 שלב 1: פתיחת Access ויצירת מסד נתונים

1. **פתח את Microsoft Access**
2. **בחר "Blank Database" (מסד נתונים ריק)**
3. **תן שם למסד הנתונים** - לדוגמה: "PhoneDialer.accdb"
4. **לחץ "Create" (צור)**

## 📋 שלב 2: יצירת הטבלאות

### דרך א: הפעלת סקריפט SQL

1. **פתח חלון SQL:**
   - לחץ על **Create** (יצירה) בסרגל העליון
   - לחץ על **Query Design** (עיצוב שאילתה)
   - סגור את חלון "Show Table" שמופיע
   - לחץ על **SQL View** (תצוגת SQL) בסרגל

2. **העתק והדבק את הקוד:**
   - פתח את הקובץ `PhoneDialerTables.sql`
   - העתק את כל הקוד מהקובץ
   - הדבק בחלון ה-SQL של Access

3. **הפעל את הסקריפט:**
   - לחץ על **Run** (הפעל) בסרגל (סמל אדום עם חץ)
   - חכה שהסקריפט יסתיים
   - שמור את השאילתה (לחץ Ctrl+S)

### דרך ב: יצירה ידנית (אם הסקריפט לא עובד)

1. **צור טבלת Contacts:**
   - לחץ **Create** → **Table Design**
   - הוסף את השדות הבאים:
     ```
     ContactID: AutoNumber (Primary Key)
     ContactName: Text (100)
     PhoneNumber: Text (20)
     Email: Text (100)
     Notes: Memo
     DateAdded: Date/Time
     LastContact: Date/Time
     IsActive: Yes/No
     ```

2. **צור טבלת SpeedDial:**
   ```
   DialID: AutoNumber (Primary Key)
   DialIndex: Number (Integer)
   ContactName: Text (100)
   PhoneNumber: Text (20)
   Description: Text (255)
   DateAdded: Date/Time
   ```

3. **צור טבלת CallHistory:**
   ```
   CallID: AutoNumber (Primary Key)
   PhoneNumber: Text (20)
   ContactName: Text (100)
   CallDate: Date/Time
   CallTime: Date/Time
   CallDuration: Number
   CallType: Text (20)
   Notes: Memo
   ```

## 🔧 שלב 3: פתיחת עורך ה-VBA

1. **פתח את עורך ה-VBA:**
   - לחץ **Alt + F11**
   - או: לחץ על **Database Tools** → **Visual Basic**

2. **הגדרות אבטחה (אם נדרש):**
   - בעורך VBA, לחץ **Tools** → **Options**
   - לחץ על **Security**
   - סמן **"Trust access to the VBA project object model"**
   - לחץ **OK**

## 📝 שלב 4: ייבוא המודולים

### ייבוא PhoneDialerModule.bas:

1. **בעורך VBA:**
   - לחץ על **File** → **Import File**
   - נווט לתיקיית הפרויקט שלך
   - בחר את הקובץ `PhoneDialerModule.bas`
   - לחץ **Open**

### ייבוא הטופס:

1. **צור טופס חדש:**
   - בעורך VBA, לחץ על **Insert** → **UserForm**
   - תן לטופס את השם: `PhoneDialerForm`

2. **העתק את הקוד:**
   - פתח את הקובץ `PhoneDialerForm.frm`
   - העתק את כל הקוד מהקובץ (מלבד החלק הראשון עם VERSION ו-BEGIN)
   - הדבק בחלון הקוד של הטופס

3. **עצב את הטופס (אם נדרש):**
   - גרור פקדים מ-Toolbox
   - צור את הפקדים הבאים:
     - Label לתצוגת מספר (lblNumberDisplay)
     - 12 כפתורים למקשים (btnNumber0-btnNumber9, btnStar, btnHash)
     - 4 כפתורי שליטה (btnCall, btnHangUp, btnClear, btnBackspace)
     - 5 כפתורי חיוג מהיר (btnSpeedDial1-btnSpeedDial5)
     - ListBox לאנשי קשר (lstContacts)
     - ListBox להיסטוריה (lstHistory)
     - Label לסטטוס (lblCallStatus)

## 🎨 שלב 5: עיצוב הטופס (אופציונלי)

אם הטופס לא נפתח אוטומטית:

1. **פתח את מצב עיצוב הטופס:**
   - בחר את הטופס PhoneDialerForm
   - לחץ על הכפתור הימני → **View Object**

2. **הוסף פקדים:**
   - מה-Toolbox, גרור פקדים לטופס
   - שנה שמות בחלון Properties

3. **דוגמה לפקדים נדרשים:**
   ```
   lblNumberDisplay: Label, Caption="הזן מספר טלפון"
   btnCall: CommandButton, Caption="חיוג"
   btnClear: CommandButton, Caption="ניקוי"
   lstContacts: ListBox
   ```

## 🚀 שלב 6: הפעלת החייגן

### דרך א: הפעלה ישירה מהקוד

1. **חזור ל-Access (חלון ראשי)**
2. **צור מאקרו חדש:**
   - לחץ **Create** → **Macro**
   - בחר **RunCode**
   - בשדה Function Name, הקלד: `ShowPhoneDialer`
   - שמור את המאקרו כשם: `OpenDialer`

3. **הפעל את המאקרו:**
   - לחץ פעמיים על המאקרו
   - החייגן ייפתח

### דרך ב: הפעלה מחלון מיידי

1. **פתח חלון מיידי:**
   - לחץ **Ctrl + G**

2. **הקלד את הפקודה:**
   ```vba
   ShowPhoneDialer
   ```

3. **לחץ Enter**

## 🔧 שלב 7: בדיקת תקינות

### בדיקת טבלאות:
1. **בחלון ראשי של Access, לחץ על Navigation Pane**
2. **ודא שרואים את הטבלאות:**
   - Contacts
   - SpeedDial  
   - CallHistory

### בדיקת קוד:
1. **פתח את עורך VBA (Alt+F11)**
2. **ודא שרואים ב-Project Explorer:**
   - PhoneDialerModule
   - PhoneDialerForm

### בדיקת הפעלה:
1. **הפעל את הפקודה:** `ShowPhoneDialer`
2. **אם יש שגיאה, בדוק:**
   - שההגדרות אבטחה מאפשרות הפעלת מאקרו
   - שכל הקבצים יובאו כראוי

## 🚨 פתרון בעיות נפוצות

### בעיה: "Security Warning"
- **פתרון:** לחץ **Enable Content** כשהאזהרה מופיעה

### בעיה: "Can't find project or library"
- **פתרון:** בעורך VBA, לחק **Tools** → **References** ובדוק שכל הספריות מסומנות

### בעיה: הטופס לא נפתח
- **פתרון:** בדוק שהקוד הועתק נכון לטופס
- **פתרון 2:** צור טופס חדש והעתק שוב

### בעיה: שגיאות קומפילציה
- **פתרון:** בדוק ש-Option Explicit קיים בראש כל מודול
- **פתרון 2:** לחץ **Debug** → **Compile** ותקן שגיאות

## 📞 איך להשתמש בחייגן

1. **הזן מספר טלפון** באמצעות המקשים
2. **לחץ על "חיוג"** לביצוע החיוג
3. **השתמש בחיוג מהיר** עם הכפתורים 1-5
4. **נהל אנשי קשר** דרך הרשימה
5. **צפה בהיסטוריה** ברשימת השיחות

- **שמור תכופות** - שמור את מסד הנתונים בכל שלב
- **גבה נתונים** - צור עותק גיבוי לפני שינויים
