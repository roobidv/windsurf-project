# עדכון מודולים ב-Access בלי Import (שומר עברית)

## למה צריך את זה
ייבוא (`Import File`) של קבצי `.bas` לפעמים הורס עברית בגלל קידוד/Code Page.
הפתרון כאן קורא את הקבצים כ-UTF-8 ומעדכן את המודולים ישירות דרך ה-VBE.

## דרישות מקדימות (פעם אחת)
1. ב-Access: `File -> Options -> Trust Center -> Trust Center Settings -> Macro Settings`
2. לסמן: **Trust access to the VBA project object model**
3. לסגור ולפתוח את Access מחדש.

## התקנה (פעם אחת)
במקום Import, עושים **העתק/הדבק** של המודול הבא לתוך Access:
- קובץ: `Access-Projects\VbaModuleUpdater.bas`
- ב-VBE: Insert -> Module
- להדביק את כל הקוד
- לשמור

## שימוש
1. פתח חלון Immediate: `Ctrl+G`
2. הרץ:
   ```vba
   UpdateAllDialerModules
   ```
3. תתבקש להזין נתיב לתיקיית `Access-Projects`.
   - דוגמה: `C:\Users\USER\Dropbox\VB6\VBA\CascadeProjects\windsurf-project\Access-Projects\`
4. בסיום: לבצע `Debug -> Compile`.

## מה זה מעדכן
הפקודה `UpdateAllDialerModules` מעדכנת את המודולים:
- `DatabaseUtilities`
- `InsertDataModule`
- `CompleteSetup`
- `PhoneDialerWorking`

## הערות
- זה **לא נוגע** ב-`PhoneDialerSimple`.
- אם אחד המודולים לא קיים בפרויקט, תקבל הודעת שגיאה עם שם המודול והקובץ.
- אם הקבצים נשמרו לא כ-UTF-8, אפשר לשנות ב-VSCode: `Save with Encoding -> UTF-8`.
